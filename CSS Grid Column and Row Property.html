Grid Child Elements(Items)

A grid container contains grid items.

By default, a container has one grid item for each column, in each row, but you can style the grid items so that they will span multiple columns and rows.


The grid-column property defines on which column(s) to place an item. You define where the item will start, and where the item will end.

To place an item, you can refer to line numbers, or use the keyword "span" to define how many columns the item will span. For Both grid-column and grid-row
